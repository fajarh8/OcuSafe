Help

Aplikasi ini merupakan aplikasi offline sehingga tidak terhubung dengan media lain. Developer aplikasi dan pihak ke-3 tidak dapat mengambil data melalui aplikasi tanpa persetujuan langsung dari user. Informasi user bersifat anonim sehingga terjamin kerahasiannya.

Ketik angka yang diinginkan pada kotak setting time kemudian klik tombol on.

Pencet tombol yang berada pada kotak setting time hingga berubah warna (sudah off).

Anda dapat menghapus aplikasi melalui sistem.
